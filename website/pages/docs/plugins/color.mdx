# Color

```bash
yarn add @elodin/plugin-color
```

The color plugin converts all used colors into a consistent color format.

## Usage

To use the plugin, simply add it to your Elodin configuration and you're ready to go.

```javascript name=elodin.config.js
import color from '@elodin/plugin-color'

module.exports = {
  plugins: [color()],
}
```

## Configuration

| Option | Type                           | Default | Description                      |
| ------ | ------------------------------ | ------- | -------------------------------- |
| format | `rgb`, `hsl`, `hex`, `keyword` | `rgb`   | The prefered output color format |

### Example

Let's say we want all colors to be written as `rgb` colors.

```elodin name=button.elo info=Input
style Button {
  backgroundColor: rgb(0 10 0)
  borderTopColor: white
}
```

```elodin name=button.elo info=Output
style Button {
  backgroundColor: rgb(0 10 0)
  borderTopColor: rgb(255 255 255)
}
```
