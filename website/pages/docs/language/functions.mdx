# Functions

## Percentage

Percentages are used for values that represent percentage values and accept a number between 0 and 100.

```elodin name=button.elo
style Button {
  width: percentage(50)
}
```

## Color

```elodin name=button.elo
style Button {
  borderTopColor: rgb(100 0 100)
  borderLeftColor: rgba(100 0 100 percentage(50))
  borderBottomColor: hsl(250 percentage(50) percentage(20))
  borderRightColor: hsla(250 percentage(50) percentage(20) percentage(50))
}
```

## Math

```elodin name=button.elo
style Button {
  width: add(500 50)
  # can also be nested
  height: add(300 mul(5 sub(10 2)) 200)
}
```

## RawValues

RawValues are special values used to bypass the type-check to pass platform-specific values.<br />
They're usually used with [Raw Properties](/docs/language/styles#raw-properties) and should be used with caution as they might not translate to other targets.

```elodin name=button.elo
style Button {
  __transition: raw("all 200ms linear")
}
```
